## Setup

1. Add your resume files to the project directory:
   - Add your resume PDF as `resume.pdf` (will download as `Khushi_Sirohi_Resume.pdf`)
   - Add your resume DOCX as `resume.docx` (will download as `Khushi_Sirohi_Resume.docx`)
   
   **Note:** The download buttons are configured to work with files named `resume.pdf` and `resume.docx`. Simply place your resume files in the same directory as `index.html` with these exact names.

2. Update contact information in `index.html`:
   - Email address
   - GitHub profile URL
   - LinkedIn profile URL

3. Customize the content:
   - Update experience details
   - Modify skills and technologies
   - Adjust any personal information
